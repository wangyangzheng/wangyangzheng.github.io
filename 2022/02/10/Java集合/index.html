Java集合详解

发表于 2022-02-10 分类于 Java

Java集合

1.集合中只能存储对象,比如将1存入集合，会将1封装为integer类型

2.集合中存放的依然是对象的引用，对象本身依然是存储在堆内存中

3.集合可以存放任意类型

集合的分类

set,map,list

set集合

1.hashset

1.不可重复存

2.线程不安全

3.元素存储的地方随机

值存在的地方由hashcode值决定，重复的含义是hashcode是否相等equal()

HashSet实现的是set接口，set接口继承了collection接口

  • hashset的使用
  • 遍历集合
  • 两个对象相等的条件

hashcode()方法返回值相等

equals()方法比较相等

2.treeset

TreeSet是SortedList接口的实现类，TreeSet可以保证集合元素处于排序状态，TreeSet支持两个排序方式：自然排序，定制排序，默认情况下，采用自然排序

- 遍历
  • 自然排序的实现方法

compare To(Object obj)

list集合

1.ArrayList

list代表一个元素有序,且不可重复的集合，集合中每个元素都有其对应的索引

list允许使用重复元素，可以通过索引来访问指定位置的集合元素

list默认按照元素添加的顺序设置元素的索引

list集合中添加了一些根据索引操作集合的方法

  • 常见方法

1.void add(下标，元素)；

map集合

map就是键值对，一个键，对应着一个数据

map中的key和value都可以是任何引用类型的数据

Map中的键不可以重复

key和value之间存在单向的一一对应关系，通过指定的key总找到唯一的value
